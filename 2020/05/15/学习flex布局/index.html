学习Flex布局

两个重要概念

什么是flex container和flex items？
开启flex布局的元素叫做flex container
flex container的子元素叫做flex items

content就叫做flex container，box就叫做flex items

如何成为flex container

给content设置display为flex或是inline-flex
设置display:flex是块级元素，设置display:inline-flex是行内元素

flex布局模型

基础概念需要了解明白
主轴 main axis 交叉轴 cross axis
主轴开始位置 main start 主轴结束位置 main end
交叉轴开始位置 cross start 交叉轴结束位置 cross end
主轴长度 main size 交叉轴长度 cross size

flex相关属性

常用的属性

作用在flex container上的css属性

flex-direction

注意：当flex container开启了flex布局(也就是设置了display为flex)，里面的flex items会默认沿着main axis(主轴)从main start到main end方向排布
flex-direction决定主轴的方向
它有四个值
1.row，默认值，主轴从左到右
2.row-reverse，主轴从右到左
3.column，主轴从上到下
4.column-reverse，主轴从下到上

justify-content

justify-content是决定flex items在main axis(主轴)上的对齐方式
flex-start从主轴开始地方对齐
flex-end从主轴结束地方对齐
center主轴中间对齐
space-between
最外面的items两端无距离，中间距离等分

space-evenly
items左右距离全部等分

space-around
最外面的items两端距离是中间距离的一半

align-items

align-items是决定一行items在cross axis(交叉轴)上的对齐方式
normal或stretch
在items没有设置height时，会使items的高度填充整个flex container
items设置了高度后，这条属性值不会有任何效果
flex-strat在交叉轴的开始地方对齐

flex-end在交叉轴的尾部地方对齐

center在交叉轴上居中对齐

baseline对于基线对齐
它会以第一个item的基线为标准

flex-wrap

flex-wrap决定了flex container是单行还是多行
假如items有8个宽度为100px，外面的container的宽度为500px，此时显示出的样子应该是items的宽度被压缩，8个items均在一行显示

nowrap默认值，单行显示，如上图显示的样子
wrap多行显示，如下图

warp-reverse也是多行显示，但是对比上图是将交叉轴反转显示的样子

flex-flow

flex-flow就是flex-drirection和flex-warp的简写
flex-flow: wrap row-reverse;
顺序任意

align-content

align-content是在多行时，每行在cross axis(交叉轴)上的对齐方式
justify-content的控制对象是每一个items而align-content的控制对像是每一行，两个属性值差不多，用法也类似
stretch默认值
在items没有设置高度的时候，items会自己进行拉伸，因为又规定了多行，所以如图所示

flex-start每一行从交叉轴开始的地方对齐

flex-end每一行从交叉轴结束的地方对齐

center在交叉轴中间对齐

space-between
最上和最下的两行items紧贴container，中间每行距离等分

space-evenly
每行items的上下距离全部等分

space-around
最上和最下的两行items到container的距离是中间距离的一半

注意：一般都是多行的时候使用该属性

作用在flex items上的css属性

order

order可以决定items的排布顺序
可以设置任意整数，数值越小越排在前面，默认值为零
这个属性用的比较少，知道有这个属性即可

align-self

可以通过单个items的设置align-self，覆盖container中设置的align-items，实现单个items在交叉轴上对齐方式的改变
auto(默认值)，默认遵从container里align-items的设置
stretch，flex-start，flex-end，center，baseline，效果与align-items中一样

flex-grow

可以让items的size扩展
当所有items的总size小于主轴的长度时，在横向来看就会有剩余的空间
如果想要items填满剩余的空间，可以使用flex-grow属性

  1. 当所有items的flex-grow值a相加大于一的时候
    每个items扩展的尺寸为 剩余size*grow值/a
    此时横向肯定是撑满的
  2. 当所有items的flex-grow值a相加小于一的时候
    每个items扩展的尺寸为 剩余size*grow值
    此时横向是不一定撑满的，横向还是会有剩余的size

注意：items扩展后的最终size不能超过max-width或max-height

flex-shrink

该属性可以让items的size收缩
当所有items的总size大于container的size，就会进行自动收缩
默认每个items中都有设置flex-shrink：1
同上，属性值总和大于一时，会按照各自比例进行收缩，属性值总和小于一的时候，按照各自比例收缩后会有溢出的情况
注意：items扩展后的最终size不能超过min-width或min-height

flex-basis

flex-basis用来设置items在主轴方向上的base size(基础大小)
auto为默认值、具体数值(100px)
最终决定flex items的size大小的因素，优先级
1.max-height/max-width/min-widt/min-height
2.flex-basis
3.width/height
4.内容本身大小
