学习Grid布局笔记

就是将页面划分成一个个的格子，任意组合不同的格子做出想要的布局样式

重要概念

1. 容器和项目
在一个区域设置display：grid后，该区域就是一个容器（container），容器内部采取网格定位的子元素称为项目（item）
注意：项目一定是容器中的顶层子元素，项目中的子元素不是项目
2. 行和列
容器中的垂直区域称为“列(column)”,水平区域称为“行(row)”
3. 单元格
垂直区域和水平区域相交叉的区域称为“单元格(cell)”
4. 网格线
划分网格的线叫做“网格线(grid line)”
水平网格线划分出行，垂直网格线划分出列
五行就有六根水平网格线，五列就有六根垂直网格线

Grid属性分为两类，一类在容器上，叫容器属性，一类在项目上叫做项目属性

容器属性

display:grid

指定该区域为网格布局，也可以设置成inline-grid

grid-template-column、grid-template-row

用来定义每一列的宽度，和每一行的高度

上面代码就是定义了三行三列的网格，宽高分别为100px的格子
可以使用绝对单位px，也可以使用百分比

1. repeat()

重复写麻烦，可以使用repeat()函数
第一个参数是重复次数，第二个参数是重复的数值
三行三列可以表达成

repeat(2，100px 20px 100px)这个定义了六列

2. auto-fill

格子的大小确定了，但容器大小没有确定，想要每一行(列)尽可能容纳多的格子，可以使用关键字suto-fill，表示自动填充
repeat(auto-fill,100px)

3. fr

表示比例关系，2fr是1fr的2倍
grid-template-columns: 1fr 2fr;

4. auto

表示自动大小
grid-template-columns: 12% auto 12%;

行间距，列间距

grid-column-gap是设置列间距
grid-row-gap是设置行间距
grid-gap是设置行和列的简写方式
grid-gap:<行><列>;省略第二个值后，会被认为第二个值等于第一个值

grid-template-areas

用于指定区域

某些区域不需要使用就用点替代

grid-auto-flow

网格分好后会按照顺序安放子元素，默认顺序是先行后列
grid-auto-flow是用来决定安放顺序的
grid-auto-flow:column就是先列后行

单元格中所有内容位置

justify-items决定单元格内容的水平位置
align-items决定单元格内容的垂直位置
属性值分别有：

  • start向上对齐
  • end向下对齐
  • center居中
  • stretch充满

简写形式：place-items:<垂直><水平>

容器中内容区域的位置

整个内容区域在容器里面的位置
justify-content
align-content
属性值同上
place-content:<垂直><水平>

项目属性

指定项目位置

就是指定项目的边框在那一条网格线上

  • grid-column-start:左边框所在位置
  • grid-column-end:右边框所在位置
  • grid-row-start：上边框
  • grid-row-end：下边框

简写形式：

  • grid-column:
  • grid-row:

grid-area

指定项目放在哪一区域

justify-self和align-self

和justify-items、align-items用法一样，但只作用于单元格
